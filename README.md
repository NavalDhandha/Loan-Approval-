# Loan-Approval-
This is project for financial institution for loan approval automation 

Loan approval Automation
Context:
A financial institution that provides personal loan products. At present loan approvals are done manually, and team of analysts is tasked to streamline this process by automating it. To achieve this, team is gathering data on loan approvals from the past years. Aim is to extract valuable insights and establish rules from the past data that can be used to automate the approval process. 
Task 1: Visualize the data
CIBIL distribution between approved and rejected loans
Education distribution between approved and rejected loans
Income distribution between approved and rejected loans

Task 2 : Is there a relationship between income and loan amount for approved loans? Is loan amount directly proportional to annual income? 

Task 3 : What specific factors can be added that influence the decision what is the reason behind this choice? And create a model that can be used to predict the loan outcomes
